Sync UI offers sophisticated table components with built-in animations, sorting, pagination, and multiple visual variants.

## Features

- Sortable columns
- Row selection with checkboxes
- Pagination with customizable rows per page
- Expandable rows for additional details
- Animated row transitions
- Progress bars and status badges
- Responsive design
- Theme-aware styling

## Variants

### Modern

The default table variant with full features including row selection, sorting, and animated transitions.

**Features:**
- Select all/individual rows
- Sortable columns with animated indicators
- Status badges with color coding
- Animated progress bars
- Pagination controls
- Hover effects

### Minimal

A simplified table variant focused on essential data display with smooth animations.

**Features:**
- Clean, minimal design
- Animated row entrance
- Status badges
- Progress indicators
- Pagination support
- Hover effects

### Expandable

A table variant with expandable rows for displaying additional details.

**Features:**
- Expandable rows with collapse animation
- Action buttons (View, Filter)
- Detailed information grid
- Smooth expand/collapse transitions
- Icon rotation on expand
- Pagination support
