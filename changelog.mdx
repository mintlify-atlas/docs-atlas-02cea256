<Accordion title="🔒 Dec '25 – Security Update & Motion Migration">

**Release:**  
→ **v0.1.3** _(latest)_ – Security, performance & stack upgrades  
→ **v0.1.2** – Security & performance update  
→ **v0.1.1** – Security update

**Security & Stack Updates:**  
→ Next.js **16.0.0 → 16.0.7 → 16.0.10**  
→ React Server Components fixes (DoS & source exposure)  
→ Core upgrades: **MUI v7, React 19, Emotion**

**Animation Migration:**  
→ Replaced **framer-motion** with **[motion/react](https://motion.dev/docs/react)**

**Why this matters:**  
→ 🚀 Smaller bundles & faster load times  
→ 🎯 Smoother, more accurate animations  
→ 🧠 Cleaner architecture with fewer abstractions  
→ ⚡ Noticeably faster UI interactions

All components now use Motion's modern animation primitives.

**New Component:**  
→ **[Skeletons](/components/skeletons)** – Lightweight loading placeholders for cards, lists, text, and content blocks

**Documentation Improvements:**  
→ **PackageManagerTabs** (npm, yarn, pnpm, bun)  
→ One-click copy with syntax highlighting  
→ Cleaner setup & improved DX

**Coming Soon:**  
→ Block-based UI sections  
→ Copy-paste layout blocks  
→ Faster bundles & smarter tree-shaking

</Accordion>
