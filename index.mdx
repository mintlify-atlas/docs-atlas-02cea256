    <Step title="Install dependencies">
      Install Material UI, Motion, and other required packages for your framework.
      
      <CodeGroup>
      ```bash npm
      npm install @mui/material @emotion/react @emotion/styled motion
      ```
      
      ```bash yarn
      yarn add @mui/material @emotion/react @emotion/styled motion
      ```
      
      ```bash pnpm
      pnpm add @mui/material @emotion/react @emotion/styled motion
      ```
      </CodeGroup>
      
      <Note>
        See the [Installation](/installation) guide for framework-specific setup instructions.
      </Note>
    </Step>
    
    <Step title="Copy a component">
      Browse the component library, copy the code, and paste it into your project. All components are fully customizable.
      
      ```jsx
      import { Button } from "@mui/material";
      import { motion } from "motion/react";

      const MotionButton = motion.create(Button);

      export default function MyButton() {
        return (
          <MotionButton
            variant="contained"
            whileHover={{ scale: 1.05 }}
            whileTap={{ scale: 0.95 }}
          >
            Click me
          </MotionButton>
        );
      }
      ```
    </Step>
    
    <Step title="Customize and ship">
      Adjust colors, animations, and styling to match your brand. Every component supports MUI's theming system and custom styling.
      
      <Tip>
        Check out the [Story](/story) to learn about the journey behind Sync UI and how it evolved.
      </Tip>
    </Step>
